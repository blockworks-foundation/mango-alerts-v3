# Mango Alerts Server

Backend logic for the mango alerts provider

## Usage

Run `yarn` to install dependencies

```bash
yarn
```

An example `.env` file -

```
DB_USER=<DB_USER>
DB_PASS=<DB_PASS>
DB_CLUSTER=<DB_CLUSTER>
DB=<DB>
PORT=3010
MAIL_USER=<MAIL_USER>
MAILJET_KEY=<MAILJET_KEY>
MAILJET_SECRET=<MAILJET_SECRET>
```
